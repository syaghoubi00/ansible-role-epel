# EPEL

An Ansible role to install EPEL

## Requirements

1. A distro that is capable of using EPEL (RHEL, CentOS, Alma, Rocky, etc).
2. EL Version 9

## Role Variables

None.

## Dependencies

None.

## Example Playbook

```yaml
- hosts: epel
  roles:
    - syaghoubi00.epel
```
